# XTool MetalFab Addons

Every xTool MetalFab owner will eventually run into these problems. This repository contains the solutions.
Open-source add-ons, fixtures and workflow improvements for the xTool MetalFab.

Examples include:
* Smart slats
* Tube supports
* Precision stops
* Rotary fixtures
* Pass-through tables
* Oversized workholding solutions

Some are finished.
Most are prototypes.
A few future problems already fixed.
---

## Current Projects

### Slat-01 Flat Rail

**Problem**

Rectangular tube stock rocks on the standard zig-zag slats.

**Solution**

Drop-in flat support rails that provide a stable and repeatable support surface.

**Status**

Planning
